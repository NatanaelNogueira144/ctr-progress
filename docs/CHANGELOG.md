## [0.1.0] - 2026-07-20

### Added

- Initial release of **CTR Progress**.
- Browser-based overlay designed for use as a narrow side panel in OBS.
- Support for tracking personal bests, goals, and world records.
- Support for tracking current grind metrics for the selected setup.
- Selection flow based on mode, track, restriction, and engine.
- Persistent local data storage using browser `localStorage`.
- `/goal-progress` page with progress bars showing completed goals.
- Compact UI intended for speedrun practice and livestream layouts.
